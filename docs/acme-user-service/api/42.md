# Users API

## GET /users/:id

Retrieves a single user by internal ID.

**Authentication:** Bearer token with active org session required.

**Path parameters**

ParameterTypeDescriptionidstringInternal user ID (cuid)

**Response — 200 OK**

```json
{
  "id": "clxyz123abc",
  "email": "alice@example.com",
  "role": "admin",
  "createdAt": "2025-01-15T10:30:00.000Z"
}
```

**Error responses**

StatusCondition401Missing or invalid Bearer token403Active org session not in token404User not found or belongs to another org

## POST /users/:id/deactivate

Deactivates a user account. Admin role required.

**Request body**

```json
{
  "reason": "Employee offboarding"
}
```

**Response — 200 OK**

Returns the updated user object with `status: "deactivated"`.

**Error responses**

StatusCondition403Caller does not have admin role404User not found409User is already deactivated
